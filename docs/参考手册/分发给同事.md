# 分发给同事

> 参见 [`README.md`](../../README.md)。本工具已是**自包含包**（语义文档、模板、两份通用 Skill
> 均随包），可当普通 wheel 分发。

```powershell
pip install build
python -m build --wheel     # 产出 dist\cosmic_kb-<版本>-py3-none-any.whl
```

对方 `pip install "cosmic_kb-<版本>-py3-none-any.whl[complete]"` 后运行
`cosmic_kb skill install` 即可，不需要整仓库。离线/内网场景的整包 zip 兜底见
[`../../scripts/make_dist.ps1`](../../scripts/make_dist.ps1) + [`../../scripts/安装说明.md`](../../scripts/安装说明.md)。

发版时 `make_dist.ps1` 会按当前包版本自动生成"版本固定安装口令"写回 README 与 安装说明.md
（`INSTALL-TOKEN` 标记对之间的整块），不用人肉维护版本号——详见
[`docs/设计方案/分发与多agent接入方案.md`](../设计方案/分发与多agent接入方案.md)。

> 本篇只讲"把某个版本递给别人"。**改完包代码要发一个新版本**（改版本号 → 刷安装口令 → 打 wheel →
> 发 PyPI → 打 tag）的完整流程见 [`发版流程.md`](发版流程.md)。
